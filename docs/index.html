Research Project · ESP32 + DHT11 + MQ7

Electronic Nose for Early
Crop Disease Detection

Plants under disease stress emit Volatile Organic Compounds (VOCs) — including elevated CO — and show measurable changes in microclimate temperature and humidity days before visual symptoms appear. This system uses an ESP32 microcontroller paired with a DHT11 (temperature + humidity) and an MQ7 (CO) sensor to capture and classify these signatures in real time using a trained ML model.

ESP32 Arduino Boilerplate
Upload this to your ESP32 (set baud: 115200)
// AGRI-NOSE — ESP32 + DHT11 + MQ7
// Dashboard expects: "CO,TEMP,HUM\n"

#include "DHT.h"

#define DHT_PIN   4    // GPIO4 → DHT11 DATA
#define MQ7_PIN  34   // GPIO34 (ADC) → MQ7 AOUT
#define DHT_TYPE DHT11

DHT dht(DHT_PIN, DHT_TYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();
  // MQ7 warm-up: wait 60 seconds
  delay(60000);
}

void loop() {
  // Read DHT11
  float temp = dht.readTemperature();
  float hum  = dht.readHumidity();

  // Read MQ7 ADC → convert to ppm
  int   raw  = analogRead(MQ7_PIN);   // 0–4095
  float volt = raw * (3.3 / 4095.0);
  float co   = volt * 100.0;           // approx ppm

  // Validate DHT11 reading
  if (isnan(temp) || isnan(hum)) {
    Serial.println("ERROR");
    delay(2000);
    return;
  }

  // Format: "CO,TEMP,HUM\n" → parsed by dashboard
  Serial.print(co);    Serial.print(",");
  Serial.print(temp);  Serial.print(",");
  Serial.println(hum);

  delay(1000); // 1 Hz rate
}
⚠️ Install DHT sensor library by Adafruit from Arduino Library Manager before compiling.

ESP32 GPIO pin mapping

DHT11 VCC
Power supply
→ ESP32 3.3V
DHT11 GND
Ground
→ ESP32 GND
DHT11 DATA
Signal (add 10kΩ pull-up)
→ ESP32 GPIO 4
MQ7 — Carbon Monoxide
MQ7 VCC
Power supply (heater needs 5V)
→ ESP32 VIN (5V)
MQ7 GND
Ground
→ ESP32 GND
MQ7 AOUT
Analog output (use voltage divider!)
→ ESP32 GPIO 34 (ADC)
⚠️ Important: MQ7 AOUT outputs up to 5V but ESP32 ADC max is 3.3V. Use a voltage divider (10kΩ + 20kΩ) to step down the signal before connecting to GPIO34.
